mapskin

mapskin is a collection of scalable vector icons for geospatial. Build efficient webmapping experiences using meaningful symbols and simple CSS.

Download

mapskin 1.0

Geospatial Web Font

mapskin is the visual language for geospatial. It illustrates concepts and objects of modern cartography in a clean iconic web font.

Scalable and skinnable

mapskin icons scale to any font size or screen, and can easily be customized with CSS rules.

Free, Libre and Open Source

mapskin is yours. The Geospatial Web Font is free for commercial or personnal use, as for any other beneficial purpose.
