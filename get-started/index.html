get started

Set up and start using mapskin in few seconds.
Check out the simple examples bellow to get started.

Using the mapskin CDN

  • Add the following link tag to the <head> section of your HTML document: 
  • <link href="https://mapskincdn.appspot.com/1.0/mapskin.min.css" rel="stylesheet">
  • Add an example icon anywhere in the <body> section using the inline element bellow: 
  • <i class="ms ms-zoom"></i>
  • and enjoy !

Help build the geospatial web font

Using your own mapskin copy

  • Download and extract mapskin-1.0.zip
  • Copy the /fonts and /css folders to your working directory.
  • The /fonts folder must contain mapskin.svg, mapskin.ttf , mapskin.eot and mapskin.woff.
  • The /css folder must contain mapskin.css and/or mapskin.min.css
  • Add the following link tag to the <head> section of your HTML document: 
  • <link href="path/to/css/mapskin.min.css" rel="stylesheet">
