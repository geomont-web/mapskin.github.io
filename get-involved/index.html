Request new icons

You are invited to request new icons which may be created and included in next mapskin public release.

Open an issue on Github to do so, and please follow the steps bellow:

  • First check if a similar icon request already exists.
    If yes, please give a +1 to that one rather then creating a new issue.
  • Create a new issue if appropriate, using the proposed icon name as Title. Please always use the ms- prefix for the icon name.
  • Describe the requested icon in a comprehensible way. A sample image may also help to share your idea.

Report bugs

Find a bug in mapskin ?
Anything wrong with an icon shape, name or CSS ?

Please let us know by opening an issue on Github as described bellow:

  • First check if a similar bug report already exists.
    If yes, please confirm it by giving a +1 rather then creating a new issue.
  • Create a new issue if appropriate.
  • Entitle your new issue with a brief sentence refering to the reported bug.
  • Describe the encountered problem in comprehensible way. A link to a fiddle reproducing it may also be useful.
